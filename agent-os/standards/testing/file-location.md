# Test File Location

Colocate tests with source files.

```
src/app/index.tsx
src/app/index.test.tsx      ← colocated
src/app/navigation.test.tsx ← route-level integration test
src/components/ui/Link.tsx
src/components/ui/Link.test.tsx
```

- Name: `[filename].test.tsx`
- No `__tests__/` directories
